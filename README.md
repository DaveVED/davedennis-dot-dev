## Getting Started

After running `create-dsqr`, you should run `undefined install` from the root directory, which will enable development across all your packages.

### Test Web App

To start and validate the web app, run the following commands:

```
nix-shell
cd packages/web
undefined install
undefined run dev
```

The app will be available at [http://localhost:5173/](http://localhost:5173/).
